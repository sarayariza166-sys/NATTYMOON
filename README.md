# NATTYMOON

# 1 Planteamiento del Problema

El problema que enfrenta el negocio Naty Moon se origina principalmente en el uso de
métodos manuales para registrar el inventario, las ventas y los movimientos de productos, lo cual
aumenta la probabilidad de errores y pérdida de información. La ausencia de un sistema digital
centralizado impide consolidar los datos en un solo lugar y dificulta su actualización en tiempo
real. 

# 2 Pregunta del programa 

 ¿Cómo puede un sistema de información contribuir a mejorar la gestión del inventario,
las ventas y el control administrativo del negocio de pijamas, optimizando los procesos y
asegurando un manejo eficiente de la información?

# 3 Objetivo general

Diseñar un sistema de información que permita gestionar de forma eficiente el inventario,
las ventas y los reportes administrativos del negocio de pijamas, facilitando el control de la
información y la toma de decisiones.

# 4 Objetivos Específicos

Analizar detalladamente los procesos actuales de inventario y ventas del negocio para identificar las falencias operativas y las oportunidades de mejora que permitan orientar el diseño del sistema propuesto.

Diseñar una base de datos estructurada que organice y centralice la información relacionada con productos, proveedores, clientes y transacciones, garantizando su acceso y actualización eficiente.

Desarrollar los módulos funcionales del sistema, incluyendo la gestión de usuarios y roles, el control de ventas, el registro automatizado de entradas y salidas de inventario y la generación de reportes.

Validar el funcionamiento del sistema mediante pruebas técnicas que permitan comprobar su confiabilidad, seguridad, usabilidad y capacidad para mejorar la gestión interna del negocio.

# 5  Justificación 

La implementación de un sistema de información en el negocio Naty Moon es
fundamental para optimizar todos los procesos de gestión administrativa y operativa que
actualmente se desarrollan de manera manual. El uso de registros en papel no solo genera
pérdida de tiempo, sino también un alto margen de error debido a la desorganización, la
duplicidad de datos y la dificultad para realizar búsquedas rápidas o reportes confiables

# 6 Alcance del Problema

 El sistema abarca la gestión interna del negocio de pijamas, incluyendo el control de
inventario, registro de ventas, manejo de proveedores, clientes y generación de reportes.
Permitirá a los administradores registrar nuevos productos (con su talla, precio, tipo y
disponibilidad), así como consultar el stock y las estadísticas semanales de ventas y ganancias.
También se incluirá un módulo de control de usuarios con roles y contraseñas,
garantizando la seguridad y el acceso restringido a la información.
