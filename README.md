# Diseño de Base de Datos – [NATYYMOON]

## Descripción

Este proyecto contiene el análisis y diseño de la base de datos.  
Incluye los diagramas, casos de uso, modelo entidad-relación, modelo relacional, diccionario de datos y requerimientos del sistema.

El propósito es aplicar los conceptos aprendidos en clase sobre modelado.

---

## Objetivos

- Identificar los requerimientos del sistema.
- Crear el modelo entidad-relación.
- Convertir el modelo ER en modelo relacional.
- Organizar la información en un diccionario de datos.
- Documentar el funcionamiento del sistema.
